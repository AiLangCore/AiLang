# Project Layout

Canonical AiLang app project layout:

```text
AiLangMyApp/
├── .gitignore
├── project.aiproj
├── AGENTS.md
├── src/
│   └── app.aos
└── Assets/
    ├── images/
    ├── fonts/
    └── locale/
        └── en.toml
```

Notes:

- `AGENTS.md` is required for AI workflow rules.
- `project.aiproj` is required and points to `src/app.aos` by default.
- `.toolchain/` is generated by runtime tooling and should be gitignored.
- `Assets/` folders are optional and may be removed when unused.
- Extension boundary: `.aos` for code and `.toml` for data.
